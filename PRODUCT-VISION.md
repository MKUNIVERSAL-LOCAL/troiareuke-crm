## MVP Phase 0 — 3대 킬러 기능 (Locked 2026-04-17)

### 킬러 ① AI 피부 진단
**무엇**: 고객 얼굴 사진 1장 → AI가 피부 나이·수분·색소·모공·주름 자동 분석 → 고객 차트에 자동 기록.

**왜**: 경쟁사 전무. "AI로 관리받는 샵"이라는 차별화된 브랜드 포지션. 데이터 축적이 킬러 ②·③의 연료.

**완료 기준**:
- 방문 시 원장이 고객 얼굴 사진 1장 촬영 (모바일/웹캠)
- 30초 이내 5가지 지표 점수 + 전회 대비 변화량 표시
- 고객 차트에 자동 이력 저장

**구현 방향 (후속 리서치 필요)**:
- 3rd-party API 활용 검토: Haut.AI, Orbo.AI, Perfect Corp, Skin Analysis API 등
- 자체 모델 학습은 Phase 2 이후

### 킬러 ② Before/After 타임랩스
**무엇**: 매 방문 사진이 같은 각도·조명 기준으로 정렬 → 슬라이드쇼·GIF·애니메이션 자동 생성 → 고객과 공유 가능.

**왜**: 에스테틱의 본질은 "변화". 말로 설명하던 것을 시각화 → 고객 만족도 + 재방문 + 인스타 바이럴.

**완료 기준**:
- 사진 촬영 시 얼굴 정렬 가이드 오버레이 (다음 촬영 시 동일 각도 유도)
- 타임랩스 뷰어: 스크러버로 시간 이동
- 고객 카톡으로 공유 링크 발송

### 킬러 ③ 자동 재방문 리마인더
**무엇**: 시술 종류·고객 이력 기반으로 AI가 권장 재방문일 예측 → 임계점 도달 시 카톡으로 자동 알림.

**왜**: 에스테틱 최대 매출 누수 원인은 "재방문 이탈". 원장이 직접 관리 불가능. 자동화 시 ROI 즉시 증명 가능.

**완료 기준**:
- 시술 종류별 기본 권장 주기 설정 (관리자 커스터마이즈 가능)
- 고객 개별 방문 이력으로 주기 보정
- 임계점 도달 시 카카오톡 알림톡 자동 발송
- 재방문율 대시보드 (전월 대비)

**구현 방향 (후속 리서치 필요)**:
- 카카오톡 알림톡: 알리고·카카오 비즈메시지·SOLAPI 등 중계사 선택
- 초기엔 단순 룰 기반, 데이터 축적 후 ML로 교체

## MVP에 **포함되지 않는** 것 (스코프 보호)

아래 기능은 **Phase 1 이후**로 확실히 미룬다. MVP 기간 중 요청 들어와도 "로드맵에 있다" 응답.

- ❌ 네이버 예약 양방향 동기화 (Phase 2)
- ❌ 직원 매칭 엔진 (Phase 2)
- ❌ 멀티지점 통합 대시보드 (Phase 2)
- ❌ 카카오 챗봇 상담 (Phase 3)
- ❌ 홈케어 자동 판매 (Phase 3)
- ❌ LTV 실시간 계산 (Phase 3)
- ❌ 음성 메모 → 시술차트 자동 생성 (Phase 3)

**현재 코드(v1.0.15)에 이미 있는 고객/예약/서비스 관리는 그대로 유지** — MVP의 베이스 레이어.
