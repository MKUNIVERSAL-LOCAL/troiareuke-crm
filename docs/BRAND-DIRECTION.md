# DERMAVIA — 브랜드 방향 (오너 확정, 2026-07-22)

> 이 문서는 트로이아르케 CRM 프로그램의 상위 브랜드 방향 정본이다.
> 모든 제품 카피·마케팅·UI 문구·신기능 설계는 이 방향에 정렬한다.

## 브랜드 체계

| 층위 | 내용 |
|---|---|
| **회사** | MK Universal |
| **플랫폼** | **DERMAVIA** |
| **카테고리** | The World's First Skin Journey Management Platform |
| **슬로건** | Every Skin Has a Journey. Every Journey Needs Intelligence. |
| **브랜드 미션** | Connecting every aesthetic professional with intelligent skin journeys. (모든 에스테틱 전문가가 더 똑똑한 피부 여정을 설계할 수 있도록 연결한다.) |

## 제품 구조와의 정렬
- **지점(사용자) 화면**: 각 샵의 브랜드가 주인공 — "{샵명} CRM" 개인화 (구현됨)
- **판매 브랜드**: DERMAVIA — 어드민 콘솔·마케팅·스토어 표기에 단계적 적용
- **핵심 가치 = 피부 여정(Skin Journey)**: 고객 등록 → 상담(비컨/AI) → 시술 기록 → 사진 타임랩스 → 재방문 리마인더로 이어지는 여정 데이터가 제품의 중심
- **Intelligence**: AI 피부분석, 재방문 예측, 전 지점 데이터(NAS 중앙화)가 카테고리 주장의 실체

## 적용 체크리스트 (단계적)
- [ ] 어드민 콘솔 브랜딩 TROIAREUKE → DERMAVIA 전환 시점 결정 (상표·도메인 확인 후)
- [ ] 로그인/온보딩 화면 슬로건 노출
- [ ] 카테고리 문구("세계 최초")는 광고 실증 이슈 검토 후 대외 사용
- 참고: 네이밍 이력 `NAMING-PROPOSAL.md`
